# ♻ Smart Waste Management System

## Project Overview

Smart Waste Management System is an AI-powered waste classification application that automatically identifies different types of waste using Deep Learning and Computer Vision. The system helps improve waste segregation and promotes sustainable waste management practices.

## Problem Statement

Manual waste segregation is time-consuming and often inaccurate. Improper waste disposal affects recycling efficiency and harms the environment. This project uses Artificial Intelligence to classify waste images automatically and recommend proper disposal methods.

## Features

* Waste Image Classification
* MobileNetV2 Deep Learning Model
* Smart Bin Recommendation
* Sustainability Tips
* Analytics Dashboard
* User-Friendly Streamlit Interface

## Technologies Used

* Python
* TensorFlow

* MobileNetV2
* OpenCV

## Dataset

Waste Classification Dataset (TrashNet)

Classes:

* Cardboard
* Glass
* Metal
* Paper
* Plastic
* Trash

## Model Architecture

The project uses MobileNetV2 as the base model for image classification. Transfer Learning is applied to improve accuracy and reduce training time.

## Project Workflow

1. Upload Waste Image
2. Image Preprocessing
3. AI Model Prediction
4. Waste Classification
5. Smart Bin Recommendation
6. Sustainability Guidance
